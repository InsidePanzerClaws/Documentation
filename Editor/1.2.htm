1.2 Type of Terrain
 

After setting the general parameters, the type of terrain can be defined by "bending" the basic flat playing area. This can be done manually, using the "Altitudes" tab, or with the built-in map generators located on the "Generator" tab.

To make the map look even more realistic, you can use the "Generate Noise" option. This generates ripples on the surface. Increases per click.
