1.1 New Map
 

Click on Menu ( bottom, right) and select New from the options list). A window opens up where the general parameters of the game can be defined.
 

You have the following options:
 

  • Level name ( defines at which level the game is to be played ).
  • File name (Here the layout can be named and saved on the hard disk).
  • Map size (from the smallest - 96x 96- to the largest -256 x 256).
  • Type of surface (on which the game is to be played).
  • Sea level is the last option in this window. Here you can set how high above sea level the game is to be played.
