1.8 Starting and Assembly Points
 

  1. When all the above items etc., have been placed in the game area, it’s time to determine the starting points, i.e., the point from which each player starts his/her particular game. You must do this to be able to save the game later. The starting point for the first player (0) is automatically placed when you create a map. For other players you must place starting points manually.

  2. An APPEARANCE POINT should be placed in the vicinity of the player's base. This is place where new, ordered units appear on the map.
    Warning: Ensure that APPEARANCE POINTS are accessible for ground units. Use ALT + G combination to show the way nodes net (this net shows you passable terrain).

  3. Place ASSEMBLY POINTS near to a base. They are target points for newly arrived units . Ensure that routes between APPEARANCE POINTS and ASSEMBLY POINTS are passable (Use ALT + G combination)
