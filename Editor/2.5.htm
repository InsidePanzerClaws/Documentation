2.5 Generator
 
Altitude

Generate type I:
random terrain generating in accordance with defined parameters for Spectral Generator.

Add to Altitudes:
adds a new relief map to the current one.
Sharpness: "Bending" angle

Generate type II:
random terrain generating in accordance with defined parameters for Perlin Generator.

Invert Altitudes:
changes the relief map.

 

Generate Noise:
Random generation of terrain (increases per every mouse click).

Generate Textures:
generates textures automatically.
   

Undo:
reverses the last operation and returns back to the previous step.
